# 🎵 Ocarina of Time — 시간의 오카리나 웹앱

브라우저에서 바로 연주할 수 있는 **젤다의 전설: 시간의 오카리나** 오카리나 웹앱입니다.
마우스 · 터치 · 키보드로 하이랄의 명곡들을 직접 연주하고, 12곡 전곡의 **악보**(버튼 탭 + 오선보)를 함께 제공합니다.

A playable *Ocarina of Time* web app — perform the songs of Hyrule with mouse,
touch, or keyboard, complete with sheet music for all 12 melodies.

## ✨ 기능 (Features)

- 🎼 **오카리나 연주** — 5개의 음(A · ▼ · ▶ · ▲ · ◀)을 Web Audio API로 합성.
  사인파 + 비브라토 + 리버브로 실제 오카리나 같은 부드러운 음색을 구현했습니다.
  **버튼/키를 누르고 있으면 그 음이 계속 지속**되어 장음(긴 음)도 연주할 수 있습니다.
- 🎨 **오카리나 이미지** — 파란 "시간의 오카리나"를 SVG로 그렸고, 음을 연주하면
  해당 구멍이 그 음의 색으로 빛납니다.
- 📜 **전곡 악보 (박자 포함)** — 12곡 모두 **버튼 탭(화살표)** 과 **오선보** 로 표기.
  오선보에는 **음표 길이(8분·4분·2분·점2분음표)** 와 곡별 **빠르기(♩=BPM)** 가 반영되어
  있어, 같은 음이라도 음표 모양과 마디 간격으로 박자를 알 수 있습니다.
- ▶ **자동 연주(Play)** — 곡별 **고유 템포와 박자대로** 자동 연주하며 버튼/악보를 하이라이트.
- ✎ **연습 모드(Practice)** — 다음에 눌러야 할 버튼을 짚어주는 가이드 연습 + 진행도 표시.
- ✨ **곡 인식** — 자유 연주 중 아는 멜로디를 치면 자동으로 알아맞힙니다
  ("♪ You played Saria's Song 사리아의 노래 ♪"). 자유 연주 중에는 **추가 효과음 없이**
  내가 친 음만 들립니다. 곡에 따라 **세상이 반응**합니다(낮/밤 전환, 비, 빛기둥).

### 🎼 전체 연주 모드 (Full-range mode)
- 악기 패널의 모드 토글로 **젤다 5음 ↔ 12홀 전체 음역(라4–파6, 반음 포함 21음)** 전환
- 피아노형 건반(마우스·멀티터치·컴퓨터 키보드)으로 연주, **누르는 동안 지속**
- 연주하는 음의 **12홀 운지표가 실시간 표시** — 앞면은 사진 위에, 보이지 않는 두 엄지홀은
  오른쪽의 작은 뒷면 모식도에 막음·열림·반홀 상태로 함께 표시

### 🌍 세계의 연주곡집 (Repertoire)
- 저작권 만료·전래곡만 모은 **퍼블릭 도메인 레퍼토리** — 비행기·반짝반짝 작은 별·
  환희의 송가·아리랑·고향의 봄·석별의 정·어메이징 그레이스·오! 수재너·스카버러 페어 등
- 곡마다 **▶ 듣기**(건반·계이름 하이라이트)와 **✎ 연습**(다음 음이 건반에서 반짝임) 제공,
  완주하면 ✔ 도장과 도전과제로 기록

### 🎓 배우기 — 오카리나 정규 레슨 (Learn)
- **실제 오카리나 교습 순서를 따르는 전공 커리큘럼 8강**: 악기의 이해(공명 원리·12홀) →
  자세와 호흡(복식호흡·롱톤) → 아티큘레이션(텅잉·레가토/스타카토) → 운지와 음계(도약 포함) →
  악보 독해(초견) → 리듬과 박자 → 표현(프레이징·비브라토) → 연습곡과 수료
- 모든 강의가 **직접 연주로 통과하는 실습** — 롱톤 게이지, 텅잉 반복, 음계·도약 순서 연주,
  청음(듣고 따라하기), 연습곡 구절, 오답 해설 퀴즈
- 오선보 그림·지공 도해 포함, 강의별 진도 저장, 전 과정 수료 시 🎓 도전과제
- 앱의 수록곡은 이 과정에서 "연습곡"으로 사용됩니다

### ✍️ 작곡 스튜디오 (Compose Studio)
- **녹음** — 연주한 리듬 그대로 악보(오선보+탭)가 되는 나만의 곡 만들기 (최대 32음)
- **저장·재생·공유** — localStorage에 저장, 공유 링크로 친구에게 전달
- 🌾 **허수아비의 노래** — 게임 속 보누루처럼 **8음**을 등록하면 자유 연주에서 인식

### 🎮 미니게임 (Games)
- 🎯 **리듬 챌린지** — **젤다 5음**과 **전체 연주(12홀 음역)** 중 선택. 젤다 모드는 5개
  레인 키·방향키·A로, 전체 연주 모드는 계이름 노트 하이웨이와 21키 건반으로 연주합니다.
  두 모드의 곡 목록·랜덤 메들리·최고 기록은 서로 독립적입니다 (Perfect/Good/Miss · 콤보 · S~C 등급).
- 🧠 **멜로디 퀴즈** — 소리만 듣고 곡 맞히기 (연속 정답 기록)
- 🔁 **메아리 게임** — 들려주는 멜로디를 따라 연주, 라운드마다 한 음씩 추가

### 🏆 기록 (Progress)
- 연주 통계(연주한 음·익힌 노래·게임 기록·학습 진도) + **도전과제 13종** (달성 시 조용한 토스트)

## ⌨️ 조작법 (Controls)

| 버튼 | 키보드 | 음 (Note) |
|------|--------|-----------|
| **A** | `A` 키 | D4 |
| **▼** C-down | `↓` | F4 |
| **▶** C-right | `→` | A4 |
| **◀** C-left | `←` | B4 |
| **▲** C-up | `↑` | D5 (높은 도/re) |

## 🗺️ 수록곡 (Songs)

**기본 오카리나 곡** — 젤다의 자장가 · 에포나의 노래 · 사리아의 노래 ·
태양의 노래 · 시간의 노래 · 폭풍의 노래

**워프 곡** — 숲의 미뉴에트 · 불의 볼레로 · 물의 세레나데 ·
영혼의 레퀴엠 · 그림자의 녹턴 · 빛의 전주곡

각 곡의 버튼 순서는 원작 게임과 동일합니다.

## ▶ 실행 방법 (How to run)

별도의 빌드나 설치가 필요 없습니다. `index.html`을 브라우저에서 열기만 하면 됩니다:

```bash
# 그냥 파일 열기
open index.html          # macOS
xdg-open index.html      # Linux
start index.html         # Windows

# 또는 간단한 로컬 서버로
python3 -m http.server 8000   # http://localhost:8000
```

> 소리는 브라우저 정책상 **첫 클릭/키 입력 이후** 재생됩니다.
